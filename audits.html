# Open Source Audit Database

The open source database currently contains audit reports of over 40 yield farming projects.

DefiYield conducts security auditing of yield farming smart contracts in a community-oriented way. Every user can request a free audit. The community then votes for projects needed to be checked first, and DefiYield audits the projects following the priority sequence. The audits are performed by experienced technicians and presented in a user-friendly format.

All criteria of the auditing process can be categorized in four groups: features of smart contract system, token distribution, token location and access of staked funds:

DefiYield also considers security points that are not given in smart contracts but describe general project positioning and reputation, such as developmemt history, project documentation and public behavior of project team.
